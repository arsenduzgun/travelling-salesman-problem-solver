# Travelling Salesman Problem Solver

This project visualises and solves the Travelling Salesman Problem (TSP) by computing a route that visits all points exactly once and returns to the start, then animating the resulting path on a canvas.

## Requirements

- **Docker**
- **Git**

## Clone the Project

git clone https://github.com/arsenduzgun/travelling-salesman-problem-solver.git

## Run the Application

### Build & Start

docker compose up --build -d

### Access

http://localhost:5000

### Stop

docker compose down --rmi all
